# ej-primera-letra-sin-repetir-py
Muestra la primera letra que no se repite de una cadena de caracteres.
Si todas las letras se repiten muestra "_".

Por ejemplo, si tenemos el string "mimamameama" el primer carÃ¡cter que no se repite es la i.

### Ejemplos:
- "abacabad" -> c
- "abacabaabacaba" -> _
- "abcdefghijklmnopqrstuvwxyziflskecznslkjfabe" -> d
- "bcccccccccccccyb" -> y
